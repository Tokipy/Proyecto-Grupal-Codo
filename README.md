# Robot Store "iohome" - Web Project

This is a group project developed as a web development practice, focused on building a fictional online store for robots using core front-end technologies.

## Description

The project consists of a multi-page website that simulates an online store where users can browse different robot products.
Key features include structured web pages using HTML, styling and layout with CSS, interactive elements using JavaScript.

Additionally, the project integrates an external API to perform real-time currency conversion using the Fetch API.

## Technologies Used

- HTML (multiple pages structure)
- CSS (styling and layout)
- JavaScript (interactivity and animations)
- Fetch API (external data handling)

## Purpose

This project was developed as a learning exercise to understand the structure of multi-page websites while practicing separation of technologies (HTML, CSS, JavaScript) and collaborate in a team environment.

## Project Status

Completed as a practice project. Possible future improvements include enhanced UI/UX, additional interactivity, and expanded functionality.
